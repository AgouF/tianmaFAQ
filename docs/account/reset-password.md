# 如何重置密码？

如果您忘记了密码或需要重置密码，请按照以下步骤操作。

## 通过邮箱重置

1. 前往 [登录页面](#)，点击 **「忘记密码」**
2. 输入您注册时使用的邮箱地址
3. 点击 **「发送重置链接」**
4. 检查您的邮箱（包括垃圾邮件文件夹）
5. 点击邮件中的重置链接
6. 设置新密码并确认

::: tip 密码要求
- 至少 8 个字符
- 包含大小写字母和数字
- 建议包含特殊字符
:::

## 通过手机号重置

1. 在登录页面点击 **「忘记密码」**
2. 选择 **「通过手机号重置」**
3. 输入绑定的手机号
4. 输入收到的验证码
5. 设置新密码

## 常见问题

### 没有收到重置邮件？

- 检查垃圾邮件文件夹
- 确认邮箱地址拼写正确
- 等待 5 分钟后重试
- 如仍未收到，请 [联系客服](/contact)

### 重置链接已过期？

重置链接有效期为 **24 小时**，过期后需要重新申请。

::: warning 注意
重置密码后，所有已登录的设备将自动退出，需要使用新密码重新登录。
:::
