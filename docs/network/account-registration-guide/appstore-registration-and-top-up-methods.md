# 外区Appstore注册方式以及充值方法

一.注册“港区”“美区”ID

1.登录 [https://www.icloud.com/](https://www.icloud.com/) 第一步

![image.png](/images/73e22133028f.webp)

点击创建Apple账户后填写信息

![image.png](/images/1977d2610461.webp)

![image.png](/images/69d4bc0a52a2.webp)

注册好后👇

![image.png](/images/b129052e22a3.webp)

直接到 手机App store登录你的账号 注意：可能需要登录两次才能登陆成功Appstore（下载免费的APP是不需要绑定付款方式的）

因为Shadowrocket需要花钱购买，我们还没绑支付方式，需要万事达或者Visa卡才可以，我们先去淘宝/咸鱼购买一个App store的港区礼品卡 

Shadowrocket价格为22港币我们只需要买22面值以上的就好

![d7ccc000b682078caf0a0fc8297f9c09.jpg](/images/664722f5a6b4.webp)

购买30港币的就OK

![ea74119aa3306090c2ea857f81e10755.jpg](/images/909464f13633.webp)

收到卡密后 进入 App store 然后兑换礼品码

![f92d7668430882e30671a72fd587b747.jpg](/images/9a8328e9b0a2.webp)

输入您的礼品码

![8515eb1be390ec09883d31d16b09c97a.jpg](/images/2eec2dbcbdae.webp)

直接兑换就OK，看到下面界面就是兑换成功了

![ce806f5be100374ca21373b3572dda13.jpg](/images/9d5d011009b6.webp)
