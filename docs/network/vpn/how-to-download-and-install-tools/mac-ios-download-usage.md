# MAC&IOS下载以及使用方式

## MAC&IOS最推荐的APP是小火箭“shadowrocket”

准备工作

1. AppStore外区账号（自己注册或者闲鱼购买）注册教程 [各平台账号注册篇](/network/account-registration-guide/)

1. IOS或Mac设备

登录您的外区账号 ⚠️ Mac的 AppStore需要在 系统设置>Apple账户>媒体与购买项目  中退出登录你现在所登录的AppStore账户

购买的账户🈲登录iCloud

登录好后打开AppStore 搜索”小火箭“或者 ”shadowrocket“

打开你购买VPN的网页，会找到类似于图片样式的导入方式

![](/images/8696cb2af9ad.webp)

随便选择一个，复制好链接后，打开shadowrocket，默认读取剪切板直接导入即可，若无法识别您的剪贴板，点击shadowrocket右上角”+“输入您复制的订阅链接即可

![](/images/c0c0548bfe6c.webp)
