# shadowrocket订阅链接导入

## Shadowrocket订阅链接导入指南

Shadowrocket作为iOS平台上广受欢迎的代理工具，其核心功能之一就是通过订阅链接快速获取和更新服务器节点信息。订阅链接是一种包含服务器配置信息的URL，用户只需导入一次，即可在服务商更新节点时自动同步，免去手动添加的繁琐操作。

### 如何获取订阅链接

订阅链接通常由VPN或代理服务商提供。在购买服务后，您会在用户后台或通过邮件收到一个以 `https://` 或 `ss://` 等协议开头的长字符串URL。请务必从可信的服务商处获取链接，并妥善保管，避免泄露。

### 详细导入步骤

导入订阅链接是配置Shadowrocket的关键一步，请按照以下流程操作：

1.  **打开Shadowrocket应用**：确保您已在App Store下载并安装了正版Shadowrocket应用。
2.  **进入订阅界面**：点击应用底部的 **“设置”** 选项卡（齿轮图标），然后选择 **“服务器订阅”**。
3.  **添加订阅**：点击右上角的 **“+”** 按钮。
4.  **填写订阅信息**：
    *   **URL**：将服务商提供的完整订阅链接粘贴至此栏。
    *   **备注**：可自定义一个名称，如“我的服务”，便于日后管理。
5.  **保存并更新**：点击右上角的 **“完成”**。返回“服务器订阅”列表，找到刚添加的条目，向左滑动，点击 **“更新”**。或者直接点击该条目，再点击底部的 **“更新”** 按钮。
6.  **启用节点**：更新成功后，节点列表会自动出现在首页的“服务器”列表中。选择一个延迟较低、速度稳定的节点，点击右侧的开关按钮将其启用（变为绿色）。首次连接时，应用可能会请求添加VPN配置，请点击“允许”。

### 使用技巧与注意事项

*   **定期更新**：为确保节点信息最新，建议定期在“服务器订阅”页面手动更新，或开启“自动更新”功能（在订阅设置中可找到）。
*   **链接失效**：如果更新失败，提示“无法获取订阅内容”，请检查链接是否过期，或联系服务商确认订阅状态。
*   **多订阅管理**：您可以添加多个不同服务商的订阅，方便切换。通过左滑条目可以进行删除或更新操作。
*   **安全提醒**：切勿在公开场合分享您的订阅链接，这等同于分享您的服务权限。

导入订阅并成功连接后，您就可以畅享安全的网络访问了。如果您在下载或基础操作上遇到困难，可以参考我们的详细教程：[MAC&IOS下载以及使用方式](/network/vpn/how-to-download-and-install-tools/mac-ios-download-usage)，其中涵盖了从获取应用到基础配置的全过程。

## 常见问题

### 导入订阅链接后，为什么节点没有显示？
首先，请确保您已完成“更新”操作，仅仅保存链接不会拉取节点。其次，检查网络连接是否正常。如果更新失败，请核对订阅链接是否正确无误，并确认订阅服务本身是否有效。有时需要关闭应用后重新打开。

### 如何设置Shadowrocket的自动更新订阅？
在Shadowrocket的“设置” > “服务器订阅”中，点击您已添加的订阅条目，进入编辑页面。找到 **“自动更新”** 选项，开启它并可以设置更新频率（如每12、24小时）。开启后，应用会在后台定期自动获取最新的节点信息。

### 订阅链接显示“无法解析”或“格式错误”怎么办？
这通常意味着链接地址不正确或已损坏。请尝试重新从服务商后台复制完整的链接，注意不要遗漏任何字符。部分链接可能需要通过“导入剪贴板”功能添加（如果链接已复制，打开Shadowrocket时会有弹窗提示）。如果问题依旧，请联系您的服务商获取支持。
