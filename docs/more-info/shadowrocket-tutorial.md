# 小火箭怎么用

## 小火箭入门指南

小火箭（Shadowrocket）是一款在iOS平台上广受欢迎的代理工具，它支持多种代理协议，可以帮助用户科学上网、访问全球网络资源并保护隐私。其界面简洁，功能强大，是许多苹果用户的首选工具。本文将详细介绍其基本使用方法。

### 核心功能与配置原理
小火箭的核心工作原理是在您的设备（iPhone或iPad）和代理服务器之间建立一个加密通道。您的网络流量会通过这个通道进行传输，从而实现突破网络限制或隐藏真实IP地址的目的。它支持SS/SSR、Vmess、Trojan、HTTP/Socks5等多种主流协议。

要使用小火箭，您首先需要一个可用的代理服务器节点信息。这些信息通常由服务提供商以订阅链接或手动配置参数的形式提供。小火箭本身不提供服务器，您需要自行获取可靠的节点来源。

### 详细使用步骤
1.  **获取与安装应用**：由于小火箭在中国区App Store已下架，您需要一个非中国区的Apple ID（如美区、港区ID）来登录App Store并搜索“Shadowrocket”进行购买和下载。具体获取方法可参考我们的另一篇指南：[MAC&IOS下载以及使用方式](/network/vpn/how-to-download-and-install-tools/mac-ios-download-usage)。

2.  **添加服务器节点**：
    *   **通过订阅链接添加（推荐）**：这是最方便的方法。打开小火箭，点击右上角的“+”号，选择“类型”为“Subscribe”。将服务商提供的订阅链接粘贴到“URL”栏中，然后点击右上角“完成”。小火箭会自动从该链接获取并更新节点列表。
    *   **手动配置添加**：点击右上角“+”号，根据服务商提供的节点信息，手动选择协议类型（如SS、Vmess），并逐一填写服务器地址、端口、密码、加密方式等参数。

3.  **连接与使用**：
    *   添加节点后，您会在首页看到服务器列表。点击您想连接的节点右侧的开关，即可开启代理连接。首次连接时，应用会请求添加VPN配置，您需要点击“Allow”并验证指纹或密码。
    *   连接成功后，顶部状态栏会出现VPN图标（一个小火箭标志），首页也会显示已连接状态和流量信息。此时，您的设备网络流量即通过该代理服务器传输。

4.  **高级功能设置**：
    *   **全局路由**：在“配置”页面，您可以设置路由模式。通常“代理”模式仅让被墙的网站走代理，节省流量；“全局”模式则所有流量都经过代理。
    *   **代理规则**：您可以自定义规则，实现更精细化的控制，例如指定某些网站或App直连、某些走代理。
    *   **HTTPS解密**：此功能可用于调试，但普通用户不建议开启，需安装并信任证书，存在安全风险。

### 使用注意事项与建议
*   **节点来源**：务必选择信誉良好、稳定可靠的服务提供商，避免使用来路不明的免费节点，以防隐私泄露或网络安全风险。
*   **及时更新**：关注小火箭应用的更新，新版本通常会修复漏洞并提升稳定性。同时，订阅链接也应定期更新以获取最新的节点信息。
*   **节省电量与流量**：长期开启代理可能会增加电量消耗。如果不需要，请及时关闭。使用“代理”路由模式而非“全局”模式可以有效节省国际流量。
*   **遵守法律**：请确保您的使用行为符合所在地的法律法规，将工具用于合法的学习和信息获取用途。

## 常见问题

### 小火箭连接成功但无法上网怎么办？
请按顺序排查：1）检查节点本身是否可用（尝试切换其他节点）；2）检查“设置”->“无线局域网”中是否为本WiFi配置了HTTP代理（如有请关闭）；3）尝试在小火箭的“配置”中，将“路由”模式切换为“全局”测试；4）重启小火箭应用或重启手机。

### 如何更新订阅链接里的节点？
进入小火箭，在“首页”或“配置”页面找到您的订阅分组（通常以订阅链接的备注名显示），向左滑动该分组，点击出现的“更新”按钮即可手动更新。您也可以在“设置”中开启“自动更新订阅”，设置定期更新频率。

### 小火箭在后台经常断开连接如何解决？
这通常是iOS系统为了省电而限制了后台活动。您可以尝试：前往iPhone“设置”->“通用”->“后台App刷新”，确保小火箭的开关是打开的。但这并不能完全保证，因为iOS对VPN后台运行有严格管理，最根本的方法是选择连接更稳定的节点或协议。
