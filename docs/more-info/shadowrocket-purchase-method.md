# Shadowrocket购买方法

## Shadowrocket是什么？

Shadowrocket是一款在iOS平台上广受欢迎的代理工具应用，它支持多种代理协议，如SS、SSR、V2Ray、Trojan等，可以帮助用户科学上网，访问国际网络资源。由于其功能强大且配置灵活，深受有网络访问需求的用户喜爱。需要注意的是，Shadowrocket仅在非中国区的App Store（如美区、港区等）上架，在中国大陆的应用商店无法直接搜索和下载。

## 如何购买Shadowrocket？

由于应用本身是付费的，并且需要外区Apple ID，因此“购买”通常包含两个步骤：获取外区Apple ID，以及完成应用的购买支付。

### 第一步：获取一个非中国区的Apple ID

这是最关键的一步。你需要注册一个非中国大陆地区的Apple ID账户，例如美国、香港、日本等。

1.  **准备工作**：准备一个全新的、未绑定过其他Apple ID的邮箱（如Gmail、Outlook）。确保你的设备（iPhone或iPad）已退出当前登录的中国区Apple ID。
2.  **开始注册**：在设备或电脑浏览器上访问Apple ID注册页面。**关键点在于**：在填写“国家或地区”时，选择目标地区（如美国）。填写地址和电话号码信息时，可以使用该地区的真实或生成的有效信息（网上有相关生成工具或公开信息可供参考）。
3.  **验证完成**：按照流程完成邮箱和手机验证，你就拥有了一个外区Apple ID。

关于更详细、一步步的图文指南，你可以参考我们之前发布的文章：[外区Appstore注册方式以及充值方法](/network/account-registration-guide/appstore-registration-and-top-up-methods)，里面详细介绍了如何选择地区、填写信息以及注意事项。

### 第二步：为外区账户充值并购买

拥有外区ID后，登录App Store，搜索“Shadowrocket”即可找到。但你需要有对应地区的支付方式来完成购买。

**常见的充值/支付方法有：**

1.  **购买礼品卡（Gift Cards）**：这是最主流和可靠的方式。你可以在一些可靠的电商平台（如淘宝、一些专门的外服游戏充值网站）购买美区、港区等App Store礼品卡。获得充值码后，在你的外区Apple ID账户中兑换，余额即可用于购买Shadowrocket。
2.  **绑定当地支付方式**：如果你有目标地区的信用卡或借记卡（如美国发行的信用卡），可以直接绑定到Apple ID的支付信息中。对于大多数国内用户而言，此方法门槛较高。
3.  **PayPal关联**：部分地区的Apple ID支持绑定与该地区匹配的PayPal账户。这同样需要你有对应地区的支付渠道。

**购买流程**：登录外区ID -> 搜索“Shadowrocket” -> 点击价格按钮（通常为2.99美元左右） -> 使用账户余额或绑定的支付方式完成购买。下载后，你可以在iPhone的“购买记录”中永久拥有它，即使切换回中国区ID，该应用也会保留（更新时需要切回外区ID）。

## 使用注意事项与建议

*   **账户安全**：请妥善保管你的外区Apple ID和密码。避免与他人共享，以防账户被封。
*   **切换账户**：在iOS的“App Store”中登录外区ID仅用于下载和更新应用，你可以在系统的“设置”-“[你的姓名]”中仍然保留你的中国区ID作为iCloud主账户，两者可以共存，互不影响iCloud数据（如照片、通讯录）。
*   **寻找配置**：Shadowrocket本身只是一个客户端工具，不提供代理服务器。你需要自行寻找可靠、安全的代理服务提供商，并获得相应的服务器配置信息（订阅链接或手动配置参数）导入到Shadowrocket中才能使用。
*   **合规使用**：请务必遵守当地法律法规，将工具用于合法的学习和工作用途。

## 常见问题

### 我已经有美区ID，但提示“无法完成购买”怎么办？
这通常是因为你的账户支付信息不完整。即使使用礼品卡余额购买，部分地区的Apple ID（尤其是新注册的）也可能需要验证账单地址。请尝试在苹果官网登录你的Apple ID，在“付款与配送”部分补充一个有效的当地地址（可使用免税州地址，如俄勒冈州）。补充后再次尝试购买。

### 购买Shadowrocket后，可以永久使用吗？
是的。一旦购买，该应用将永久关联到你的这个外区Apple ID。即使应用未来从商店下架（这种可能性很小），你已购买的应用依然可以在“已购项目”中重新下载。未来的大版本更新也无需再次付费。

### 除了Shadowrocket，还有其他类似应用吗？
有。在美区App Store上，类似的知名工具有**Quantumult X**、**Stash**、**Loon**等，它们功能侧重点和配置界面各有不同，也都是付费应用。你可以根据社区评价和个人喜好进行选择，购买方式与Shadowrocket完全相同。
