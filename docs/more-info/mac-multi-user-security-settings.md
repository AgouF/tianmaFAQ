# MAC多用户环境安全设置

## 为什么需要关注MAC多用户环境安全？

在家庭、办公室或教育机构中，一台Mac电脑常常由多人共用。每个用户可能拥有不同的文件、应用程序偏好和隐私数据。如果没有恰当的安全设置，用户可能会无意间访问或修改他人的私人文件，安装不受欢迎的软件，甚至导致系统设置被更改，影响所有人的使用体验。更严重的是，如果某个账户密码过于简单或管理不当，可能成为整个系统安全防线的突破口。因此，为每个用户建立独立、安全的空间，并配置恰当的权限，是保护个人隐私和维护系统稳定的基础。

## 核心安全设置步骤

### 1. 创建与管理独立用户账户
进入“系统设置” > “用户与群组”。点击左下角锁形图标，输入管理员密码解锁。通过“添加账户”为每位使用者创建一个标准账户。**务必避免多人共享一个管理员账户**。日常使用应使用标准账户，仅在进行软件安装或关键系统更改时使用管理员账户。定期检查账户列表，删除不再使用的账户。

### 2. 配置强密码与登录选项
为每个账户设置强密码：长度至少8位，混合大小写字母、数字和符号。在“用户与群组”中，选中账户，点击“更改密码”可进行设置。在“登录选项”中，建议：
*   关闭“自动登录”，确保每次开机都需验证身份。
*   开启“在睡眠和屏幕保护程序启动后要求输入密码”，时间建议设为“立即”。
*   （可选）为高权限账户（如管理员）启用“文件保险箱”全盘加密，这样即使电脑丢失，数据也无法被读取。

### 3. 精细化的文件权限与共享管理
这是多用户环境的核心。利用“访达”的“共享与权限”功能：
*   **个人文件夹**：每个用户的`/Users/用户名/`目录默认只有该用户和系统可完全访问。切勿随意更改此目录的权限。
*   **共享区域**：可以在`/Users/Shared/`目录存放需要交换的文件。建议为该文件夹设置权限，例如允许所有用户“读与写”。
*   **特定文件夹共享**：右键点击任何文件夹 > “显示简介” > 底部“共享与权限”。您可以添加特定用户，并为其分配“只读”、“读与写”或“无访问权限”。

### 4. 启用家长控制（适用于有儿童的家庭）
对于儿童或需要限制使用的账户，可以使用“屏幕使用时间”功能（位于“系统设置”中）。您可以：
*   **设定应用限额**：限制每日使用游戏、社交软件的时间。
*   **通信安全与限制**：控制可与该儿童联系的人员，并过滤敏感内容。
*   **始终允许或限制特定应用**：确保学习工具可用，同时限制不当应用。

### 5. 使用“客人用户”功能
对于临时使用者（如访客），务必启用“客人用户”。在“用户与群组”中勾选“允许客人登录到这台电脑”。客人账户在注销后，其所有文件和设置会被自动彻底删除，有效保护主用户的隐私并防止系统留下垃圾文件。

## 高级建议与最佳实践
*   **定期进行用户审计**：每隔一段时间，检查系统设置中的登录项、辅助功能权限，确保没有未知账户获得特殊权限。
*   **隔离管理员权限**：创建唯一一个高度受保护的管理员账户，其密码仅由设备管理者知晓。其他所有日常账户均为标准账户。
*   **利用“群组”管理权限**：如果有多位用户需要相同的文件夹权限（如一个项目组），可以创建一个“群组”，将用户加入，然后一次性对群组分配权限，管理更高效。
*   **保持系统更新**：确保macOS和所有应用程序保持最新，以修补可能被多用户环境利用的安全漏洞。关于系统维护，您可以参考我们的另一篇指南：[MAC安装方式](/catalog-2/directory-nesting-333/mac-installation)，了解如何安全地安装和更新系统。

## 常见问题

### 家人不小心用了我的管理员账户安装软件，怎么办？
首先，应立即更改您的管理员账户密码。然后，检查“系统设置” > “通用” > “登录项”和“用户与群组”，移除可疑的登录项或账户。最后，在“应用程序”文件夹中，检查并卸载不认识的软件。未来请务必为家人创建独立的“标准”账户供日常使用。

### 如何安全地让其他用户使用我Mac上的打印机？
您无需共享整个账户。最佳方式是确保打印机已正确安装并在您的账户下可用。然后，在“系统设置” > “通用” > “共享”中，开启“打印机共享”。在右侧列表中选择要共享的打印机，并点击“信息”按钮(i)来指定哪些用户可以使用它。

### 客人用户会看到我电脑里的文件吗？
不会。启用“客人用户”后，客人登录的是一个全新的、临时的标准账户。他们只能访问自己的临时桌面和`/Users/Shared/`文件夹，无法访问您个人文件夹（`/Users/您的用户名/`）或其他用户的私人文件。一旦他们注销，其产生的所有数据都会被自动清除。
