# MAC多用户环境安全设置

## 为什么需要关注MAC多用户环境安全？

在家庭、办公室或教育机构中，一台Mac电脑常常由多人共用。每个用户都有自己的文件、应用程序偏好设置和隐私数据。如果没有恰当的安全设置，可能会导致个人信息泄露、重要文件被意外删除或修改，甚至给整个系统带来安全风险。例如，家人可能误删你的工作文档，或者访客账户可能被用来安装不受信任的软件。因此，合理配置多用户环境不仅是管理上的需要，更是保护个人数据安全和维护系统稳定的关键步骤。

## 核心安全设置步骤

### 1. 创建与管理用户账户
进入 **系统设置 > 用户与群组**，点击左下角锁形图标并输入管理员密码进行解锁。在这里，你可以为每位使用者创建独立的账户。
*   **管理员账户**：应仅限少数完全可信的用户（如设备所有者）。管理员拥有最高权限，可安装软件和修改系统设置。
*   **普通账户**：适合日常使用者。他们可以运行应用程序和修改自己的设置，但无法影响其他用户或安装系统级软件。
*   **儿童账户**：可使用“屏幕使用时间”功能管理，非常适合家长控制。
*   **仅限共享账户**：适用于仅需访问共享文件夹的用户，无法登录图形界面。

**最佳实践**：你个人应使用一个普通账户进行日常操作，仅在使用管理员权限时临时切换或授权。这能最大程度减少误操作或恶意软件对系统的损害。

### 2. 配置牢固的登录密码与文件保险箱
为每个账户设置**高强度密码**（建议使用大小写字母、数字和符号的组合）。启用 **文件保险箱** 是全盘加密功能，能确保即使电脑丢失，他人也无法读取磁盘上的任何数据。你可以在 **系统设置 > 隐私与安全性 > 文件保险箱** 中开启。开启后，每个用户的数据都会被其登录密码单独加密。

### 3. 利用访客账户与快速用户切换
对于临时使用者，务必启用**访客账户**。该账户在每次注销后会自动清除所有数据和设置，完美保护隐私。同时，开启 **快速用户切换**（在“用户与群组”高级选项中），可以方便地在不同用户间切换而无需完全注销，但记得在离开电脑时锁定屏幕（`Control + Command + Q`）。

### 4. 精细设置文件夹权限
这是保护个人文件的核心。在访达中，右键点击你想保护的文件夹，选择 **“显示简介”**。在 **“共享与权限”** 部分，你可以：
*   移除其他用户的访问权限。
*   为特定用户或“群组”设置权限，例如“只读”、“只写”或“无访问权限”。
*   将私人文件夹移入你的个人专属文件夹（`/Users/你的用户名/`）中，这是最安全的做法。

### 5. 启用屏幕保护程序与立即锁定
在 **系统设置 > 屏幕保护程序** 中，设置一个较短的等待时间（如5分钟）。然后进入 **系统设置 > 锁定屏幕**，勾选 **“屏幕保护程序启动后立即要求输入密码”**。这是一个至关重要的习惯，能确保在你暂时离开时，他人无法访问你的会话。

## 高级管理与维护建议
定期检查 **系统设置 > 隐私与安全性 > 辅助功能/完全磁盘访问权限** 等列表，确保没有未知应用获得过高权限。对于管理员，应定期在“用户与群组”中审查账户列表，删除不再需要的账户。如果需要在多台Mac上部署相似的多用户环境，可以参考标准化的[MAC安装方式](/catalog-2/directory-nesting-333/mac-installation)流程，确保基础系统安全一致。

## 常见问题

### 家人共用一台Mac，如何防止孩子更改系统设置？
为孩子创建一个**受“屏幕使用时间”管理的账户**。在系统设置的“屏幕使用时间”中，你可以为这个账户单独设置“内容和隐私”限制，完全禁用“用户与群组”设置的更改，并限制可用的App和网站。

### 启用了文件保险箱，其他用户还能登录吗？
可以。文件保险箱是透明加密。每个用户使用自己的账户密码解密自己的数据分区。其他用户登录时，系统会用其密码解密其权限内的数据，但无法访问你账户下的加密数据。这确保了数据的隔离性。

### 如何临时给同事查看我一个文件夹里的文件，但不希望他复制走？
最安全的方法是使用**访客账户**，并将该文件夹的权限设置为“只读”。你可以将文件复制到一个专用文件夹，右键点击该文件夹“显示简介”，在权限设置中，为“客人”或“所有人”设置为“只读”。这样，对方可以打开查看文件，但无法复制、移动或删除它们。操作完毕后，注销访客账户，所有痕迹将被清除。
