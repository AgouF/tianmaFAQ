# 小火箭怎么导入订阅链接

## 小火箭导入订阅链接完整指南

小火箭（Shadowrocket）是一款在iOS设备上广受欢迎的代理工具，其核心功能之一就是通过订阅链接来获取和更新代理服务器节点。导入订阅链接是使用小火箭的第一步，也是确保网络连接正常的关键操作。下面将详细介绍几种主流且安全的导入方法。

### 方法一：通过剪贴板直接导入（最常用）
这是最快捷的方式，适用于已经复制好订阅链接的情况。
1.  **复制订阅链接**：首先，确保您已经从服务提供商处获得了完整的订阅链接，并已将其复制到设备的剪贴板。
2.  **打开小火箭**：在您的iPhone或iPad上启动小火箭应用。
3.  **进入订阅页面**：点击应用底部的 **“设置”** 选项卡，然后选择 **“服务器订阅”**。
4.  **粘贴并添加**：点击页面右上角的 **“+”** 号按钮。小火箭会自动检测剪贴板中的链接，并弹出确认窗口。您也可以手动在“URL”栏中粘贴链接。
5.  **完成添加**：在“备注”栏可以给这个订阅起一个易于识别的名字（例如“我的服务”），然后点击右上角的 **“完成”**。
6.  **更新节点**：添加成功后，回到“服务器订阅”页面，点击刚添加的订阅后面的 **“更新”** 按钮。小火箭便会从该链接获取节点信息，节点列表会出现在首页的“服务器”选项中。

### 方法二：通过扫码导入
如果订阅链接以二维码形式提供，此方法非常方便。
1.  **打开扫码功能**：在小火箭首页，点击右上角的扫码图标（一个方框中有个小点）。
2.  **扫描二维码**：将手机摄像头对准包含订阅链接的二维码。
3.  **自动跳转并添加**：扫描成功后，应用会自动跳转到“添加订阅”的界面，链接已自动填入。您只需添加备注并点击“完成”，然后记得返回更新即可。

### 方法三：通过配置文件（URL Scheme）导入
一些服务商会提供形如 `shadowrocket://add/sub://` 开头的专用配置链接。
1.  **点击配置链接**：在Safari浏览器或其他App中，直接点击服务商提供的这种特殊链接。
2.  **跳转并确认**：系统会询问是否打开“Shadowrocket”，点击 **“打开”**。
3.  **自动完成**：小火箭会自动启动并完成订阅的添加，通常会自动更新一次节点。您可以在“服务器订阅”设置中查看和管理它。

**重要提示**：
*   **及时更新**：订阅链接有时效性，定期在“服务器订阅”页面手动更新或开启“自动更新”功能，才能获取最新的节点。
*   **链接安全**：请务必从可信的服务商处获取订阅链接，避免隐私泄露。
*   **问题排查**：如果更新失败，请检查网络连接是否正常，以及订阅链接是否有效（有时链接可能过期）。可以尝试将链接粘贴到浏览器中，看是否能下载到一个文本文件来验证链接有效性。

## 常见问题

### 为什么我更新订阅后，节点列表还是空的？
出现这种情况通常有几个原因：
1.  **订阅链接失效**：您的订阅可能已过期，请联系您的服务商确认链接有效性。
2.  **网络问题**：更新时设备网络连接不畅，尝试切换网络（如从Wi-Fi切换到蜂窝数据）后再次更新。
3.  **链接复制不完整**：确保复制的链接没有头尾空格或遗漏字符。最稳妥的方法是在“添加订阅”时，手动长按URL栏进行粘贴。

### 我可以导入多个订阅链接吗？
可以的。小火箭支持添加多个订阅链接。您只需按照上述方法重复添加即可。所有订阅中的节点都会汇总在首页的“服务器”列表里。您可以在“服务器订阅”设置页面管理（编辑、删除、单独更新）每一个订阅。

### 导入订阅链接安全吗？会泄露我的信息吗？
**安全性主要取决于订阅链接的来源**。从正规、信誉良好的服务商处获取的链接是安全的，导入过程本身是本地操作。但切勿导入来源不明或免费的订阅链接，这些链接可能包含恶意配置，导致流量被监听或隐私泄露。请始终选择可信的服务提供商。
