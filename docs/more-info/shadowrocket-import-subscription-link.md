# 小火箭怎么导入订阅链接

## 小火箭导入订阅链接详细教程

小火箭（Shadowrocket）是一款在iOS设备上广受欢迎的代理工具，其核心功能之一就是通过订阅链接来获取并更新代理服务器节点。导入订阅链接是使用小火箭的第一步，也是确保你能稳定连接网络的关键。下面将详细介绍几种主流且安全的导入方法。

### 方法一：通过剪贴板直接导入（最常用）
这是最简单快捷的方式，适用于已经获取到订阅链接（通常是一串以`http://`或`https://`开头的URL）的情况。
1.  **复制链接**：首先，长按你获得的订阅链接（可能来自邮件、文档或网站），选择“复制”。确保完整复制了整个链接。
2.  **打开小火箭**：返回手机主屏幕，打开小火箭应用。
3.  **一键导入**：**当小火箭检测到剪贴板中有可识别的订阅链接时，屏幕上方会自动弹出一个提示横幅**。你只需点击这个横幅。
4.  **确认添加**：小火箭会自动跳转到服务器节点列表页面，并弹出确认窗口。点击“好的”或“确定”，应用便会开始从该链接下载节点信息。
5.  **完成**：下载完成后，节点列表里就会出现新的服务器。你通常需要手动点击选择其中一个节点，然后开启顶部的连接开关。

### 方法二：通过扫码导入
如果订阅链接被做成了二维码图片，这个方法非常方便。
1.  在小火箭应用主界面，点击右上角的“+”加号按钮。
2.  在弹出的菜单中，选择“扫描二维码”。
3.  将手机摄像头对准提供的订阅二维码。
4.  扫描成功后，会自动识别并弹出添加确认窗口，后续步骤与方法一相同。

### 方法三：手动添加订阅
如果自动识别失败，或者你想更精确地管理，可以使用此方法。
1.  在小火箭主界面，点击右上角的“+”加号。
2.  选择“类型”为“Subscribe”（订阅）。
3.  在“URL”栏中，粘贴或手动输入你的订阅链接。
4.  （可选）在“备注”栏填写一个易于识别的名字，如“我的服务A”。
5.  点击右上角的“完成”，小火箭便会开始获取订阅。

### 重要提示与注意事项
*   **链接安全**：务必从可信赖的服务提供商处获取订阅链接，不要使用来源不明的链接，以防安全风险。
*   **更新订阅**：服务器节点可能会更新、失效或变更。你需要定期更新订阅以获取最新节点。在小火箭的节点列表页面，**向下滑动屏幕即可手动更新订阅**（会出现刷新图标）。你也可以在订阅的设置中开启“自动更新”。
*   **多订阅管理**：小火箭支持添加多个订阅。你可以在“设置” -> “服务器订阅”中管理它们，进行启用、禁用或删除操作。
*   **兼容性**：请确保你使用的小火箭版本支持订阅链接格式。保持应用为最新版本是良好的习惯。
*   **获取客户端**：如果你是初次使用，需要先获取小火箭应用。具体下载和安装方法，可以参考这篇详细指南：[MAC&IOS下载以及使用方式](/network/vpn/how-to-download-and-install-tools/mac-ios-download-usage)。

成功导入订阅并连接后，你就可以根据规则设置，畅享安全、便捷的网络访问体验了。

## 常见问题

### 为什么复制了链接，小火箭没有弹出导入提示？
这可能由几个原因导致：
1.  复制的文本不是一个完整且被小火箭识别的订阅链接格式（比如中间有换行或缺少部分字符）。
2.  小火箭应用在后台被完全关闭，可以尝试重新打开一次应用。
3.  系统剪贴板权限问题，可以尝试重启小火箭或重启手机。如果始终不弹出，建议直接使用方法三（手动添加）进行添加。

### 导入订阅后，显示“更新失败”或“无法获取配置”怎么办？
这通常表示订阅链接本身有问题：
1.  **链接失效**：订阅链接可能已过期或被服务商禁用。请联系你的服务提供商确认链接有效性。
2.  **网络问题**：当前网络环境无法访问订阅链接所在的服务器。尝试切换网络（如从Wi-Fi切换到蜂窝数据）后再更新。
3.  **链接格式错误**：请再次核对链接是否完全正确，特别是长链接，容易在复制时头尾缺失。

### 可以同时导入多个订阅链接吗？如何切换？
完全可以。小火箭支持添加多个订阅。所有导入的节点会合并显示在同一个列表中。你可以通过节点名称或分组来区分它们。如果想单独管理某个订阅（如单独更新或禁用），可以进入小火箭的“设置” -> “服务器订阅”，在这里你可以看到所有已添加的订阅列表，并进行独立管理。
