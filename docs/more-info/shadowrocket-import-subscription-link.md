# 小火箭怎么导入订阅链接

## 如何在小火箭中导入订阅链接

小火箭（Shadowrocket）是一款在iOS设备上广受欢迎的代理工具，其核心功能之一就是通过订阅链接来获取和管理代理服务器节点。导入订阅链接是使用小火箭的第一步，也是连接网络服务的关键。这个过程虽然简单，但对于新手用户来说，清晰的指引至关重要。

### 详细的导入步骤

请按照以下步骤操作，即可成功将订阅链接导入小火箭：

1.  **获取订阅链接**：首先，你需要从你所购买的VPN或代理服务商那里获得一个有效的订阅链接。这通常是一个以 `http://` 或 `https://` 开头的长网址。请妥善保管，不要随意泄露。

2.  **打开小火箭应用**：在你的iPhone或iPad上找到并点击小火箭的图标，启动应用。

3.  **进入订阅管理页面**：
    *   应用启动后，你会看到底部有几个标签页，点击最右侧的 **“设置”**（齿轮图标）。
    *   在设置页面中，找到并点击 **“订阅”** 选项。

4.  **添加订阅**：
    *   在订阅页面，点击右上角的 **“+”** 加号按钮。
    *   这时会弹出添加订阅的窗口。你只需要在 **“URL”** 一栏中，粘贴你从服务商那里获取的完整订阅链接。
    *   **“备注”** 一栏可以填写一个方便你识别的名称，例如“我的主力节点”或服务商名称，这是可选的，但建议填写以便管理。
    *   其他选项如“更新间隔”等，保持默认即可。
    *   最后，点击右上角的 **“完成”**。

5.  **更新订阅并启用**：
    *   添加完成后，你会回到订阅列表页面，可以看到你刚添加的订阅。
    *   为了立即获取节点信息，请**向左滑动**你刚添加的订阅条目，然后点击 **“更新”**。或者，你也可以直接点击该订阅，进入详情页后手动点击“更新”。
    *   更新成功后，返回小火箭的主页（第一个标签页“主页”），你应该能看到从订阅中获取的所有服务器节点列表。
    *   选择一个你想连接的节点，点击右上角的开关，即可开始连接。

### 重要提示与故障排除

*   **链接有效性**：确保你粘贴的订阅链接完全正确且未被过期。如果链接失效，更新会失败。
*   **网络环境**：首次更新订阅时，请确保你的设备网络通畅。有时在特殊的网络环境下可能需要先通过其他方式连接网络才能成功更新。
*   **更新失败**：如果更新失败，请检查链接是否正确，并尝试切换网络（比如从Wi-Fi切换到蜂窝数据）后再次更新。
*   **节点管理**：成功导入后，你可以在主页对节点进行测速、排序、分组等操作，方便你选择最优线路。

## 常见问题

### 我点击更新订阅后，为什么节点列表还是空的？
这种情况通常有几个原因：1) 订阅链接本身已失效或错误，请向你的服务商确认；2) 当前网络无法访问订阅服务器，尝试切换网络环境；3) 小火箭首次使用可能需要授予网络权限，请检查iOS的“设置” > “小火箭”，确保“无线数据”选项是打开的。建议复制链接到手机浏览器中测试是否能打开（可能会显示为一串代码），如果不能，则是链接或网络问题。

### 订阅链接需要定期重新导入吗？
不需要。一旦成功导入，小火箭会根据你设置的“更新间隔”（默认通常是24小时）自动在后台更新节点信息。只有当服务商告知你订阅链接已变更，或者你手动清除应用数据后，才需要重新添加新的订阅链接。平时你只需要在节点列表中选择使用即可。

### 我可以导入多个订阅链接吗？
可以。小火箭支持添加多个订阅。你可以在“设置”>“订阅”页面添加多个不同服务商的订阅链接，并分别备注好名称。更新后，所有订阅的节点都会汇总显示在主页的节点列表中。你可以通过节点名称或分组来区分它们，方便管理多个线路。
