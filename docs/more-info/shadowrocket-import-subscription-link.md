# 小火箭怎么导入订阅链接

## 小火箭导入订阅链接的完整指南

小火箭（Shadowrocket）是一款在iOS设备上广受欢迎的代理工具，其核心功能之一就是通过“订阅链接”来获取和更新服务器节点信息。订阅链接是服务提供商提供的一个URL地址，包含了服务器配置、加密方式等所有必要信息。正确导入订阅链接，是使用小火箭科学上网的第一步。

### 准备工作
在开始导入之前，请确保你已经完成以下两步：
1.  **拥有一个有效的订阅链接**：你需要从可靠的VPN或代理服务商处购买服务，他们会提供一个专属的订阅链接（通常是一串以 `http://` 或 `https://` 开头的URL）。
2.  **已安装小火箭应用**：你需要在非中国区的App Store（如美区、港区）购买并下载安装Shadowrocket应用。如果你还没有安装，可以参考我们的另一篇详细教程：[MAC&IOS下载以及使用方式](/network/vpn/how-to-download-and-install-tools/mac-ios-download-usage)，其中包含了获取和安装应用的步骤。

### 详细导入步骤
请按照以下流程操作，整个过程通常在1分钟内即可完成。

1.  **复制订阅链接**：
    *   从你的服务商后台、邮件或用户中心找到你的订阅链接。
    *   长按链接，在弹出的菜单中选择“拷贝”，将链接复制到剪贴板。

2.  **打开小火箭并添加订阅**：
    *   在iPhone或iPad上打开小火箭应用。
    *   点击应用底部导航栏的“首页”或“服务器”标签。
    *   点击屏幕右上角的“+”加号按钮。
    *   在弹出的菜单中，选择“**从URL导入**”。

3.  **粘贴并确认**：
    *   系统会自动弹出窗口，并将你之前复制的订阅链接粘贴在URL栏中。如果没有自动粘贴，请长按输入框并选择“粘贴”。
    *   确认链接无误后，点击右上角的“完成”或“确定”。

4.  **更新服务器节点**：
    *   导入成功后，你会看到一个新的订阅分组（通常以服务商名称命名）。
    *   点击该订阅分组，进入详情页面。
    *   为了获取最新的服务器列表，请点击页面底部的“**更新**”按钮。小火箭会从订阅链接指向的地址拉取最新的节点配置信息。

5.  **选择节点并连接**：
    *   更新完成后，返回小火箭首页。
    *   在服务器列表中，选择你想连接的一个节点（通常延迟越低越好）。
    *   最后，点击首页顶部那个大大的圆形“开关”按钮，即可开启代理连接。当按钮变为绿色且显示“已连接”时，表示导入和连接成功。

### 实用技巧与注意事项
*   **定期更新**：服务商的节点信息可能会变更，建议定期进入订阅分组，点击“更新”以获取最新列表。
*   **备用链接**：有些服务商提供多个备用订阅链接，如果一个失效，可以尝试导入另一个。
*   **安全提醒**：请妥善保管你的订阅链接，不要随意分享，因为它直接关联你的账户和服务。
*   **导入失败排查**：如果导入失败，请检查：1) 网络是否通畅；2) 订阅链接是否已过期或失效；3) 是否准确复制了完整链接。

## 常见问题

### 导入订阅链接后，节点列表是空的怎么办？
这通常有几个原因：首先，请检查网络连接，并尝试再次点击订阅分组底部的“**更新**”按钮。其次，可能是订阅链接本身已失效，请联系你的服务商确认链接有效性。最后，尝试完全关闭小火箭应用后重新打开，再执行更新操作。

### 可以导入多个不同服务商的订阅链接吗？
完全可以。小火箭支持添加多个订阅。你只需重复上述“添加订阅”的步骤，导入不同服务商的链接即可。在首页的服务器列表中，你可以看到所有订阅下的节点，并自由切换使用。这对于需要备用方案或访问不同地区资源的用户非常方便。

### 订阅链接和手动添加服务器有什么区别？
订阅链接是自动化方式。你只需维护一个链接，服务商更新节点、端口、密码等信息后，你一键“更新”即可同步所有更改，非常方便。而手动添加需要你逐一输入服务器地址、端口、密码、加密方式等所有参数，且后续任何变动都需要手动修改。对于大多数用户，强烈推荐使用订阅链接方式。
