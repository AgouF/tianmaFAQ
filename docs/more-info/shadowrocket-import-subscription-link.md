# 小火箭怎么导入订阅链接

## 小火箭导入订阅链接的完整指南

小火箭（Shadowrocket）是一款在iOS设备上广受欢迎的代理工具，其核心功能之一就是通过订阅链接来获取和管理代理服务器节点。导入订阅链接是使用小火箭的第一步，也是连接网络服务的关键。下面将详细介绍几种主流且安全的导入方法。

### 方法一：通过“扫码”导入（最推荐）
这是最快捷、最不易出错的方式，尤其适合从机场服务商或朋友处获取的二维码。
1.  打开小火箭应用，确保你位于首页的“服务器”标签页下。
2.  点击屏幕右上角的“+”加号按钮。
3.  在弹出的菜单中，选择“扫描二维码”。
4.  将手机摄像头对准包含订阅链接的二维码。小火箭会自动识别并读取链接。
5.  识别成功后，应用会提示“下载成功”。此时，所有节点信息已经自动添加到你的服务器列表中。

**优点**：无需手动输入，避免链接复制错误；高效安全。

### 方法二：通过“从URL导入”手动添加
如果你获得的是一个文本形式的订阅链接（通常以 `https://` 开头），可以使用此方法。
1.  同样，在“服务器”页面点击右上角的“+”加号。
2.  在菜单中选择“从URL导入”。
3.  系统会弹出一个文本框。请**长按文本框**，选择“粘贴”，将你事先复制好的订阅链接地址粘贴进去。**强烈建议使用粘贴，避免手动输入可能产生的字符错误。**
4.  点击右上角的“完成”。小火箭会开始从该链接地址获取订阅信息。
5.  获取成功后，节点列表会自动更新。

### 方法三：通过Safari浏览器一键导入
许多服务商的网站提供了“一键导入”功能，这本质上利用了iOS的URL Scheme。
1.  使用iPhone的Safari浏览器打开提供订阅链接的网页。
2.  在网页上找到“一键导入到Shadowrocket”或类似的按钮，并点击它。
3.  系统会弹窗询问“在‘Shadowrocket’中打开此页吗？”，点击“打开”。
4.  小火箭应用会自动启动，并开始处理该订阅链接，导入节点。

### 重要提示与后续设置
*   **链接安全**：请务必从可信赖的来源获取订阅链接，避免隐私泄露。
*   **更新订阅**：服务器节点可能会更新、失效或变更。你需要定期更新订阅以获取最新节点。在小火箭的“服务器”页面，**向下滑动屏幕**即可手动刷新订阅。你也可以在“设置” > “服务器订阅”中开启“自动更新”。
*   **连通性测试**：导入节点后，建议点击某个节点，选择“测试连接”或“延迟测试”，以检查该节点的当前网络质量。
*   **开启代理**：节点导入并测试完成后，回到小火箭首页，选择一个延迟低的节点，点击顶部大大的开关按钮，即可开启代理连接。当按钮变为绿色且顶部状态栏出现VPN图标时，表示连接成功。

对于Mac用户，配置流程与理念类似，但操作界面有所不同。你可以参考我们另一篇详细的指南：[MAC&IOS下载以及使用方式](/network/vpn/how-to-download-and-install-tools/mac-ios-download-usage)，其中包含了在macOS上配置小火箭的步骤。

## 常见问题

### 为什么导入订阅后，节点列表是空的或提示失败？
这通常有几个原因：
1.  **订阅链接已过期或失效**：请联系你的服务商确认链接有效性。
2.  **网络问题**：尝试切换网络（如从Wi-Fi切换到蜂窝数据）后，再次下拉刷新订阅。
3.  **链接复制不完整**：确保复制的链接没有头尾空格，且完整无误。使用“从URL导入”时，务必使用粘贴功能。
4.  **订阅格式不支持**：极少数情况下，链接可能包含小火箭不兼容的特殊格式。

### 我可以同时导入多个订阅链接吗？
可以。小火箭支持添加多个订阅源。你只需按照上述“从URL导入”的方法，多次添加不同的订阅链接即可。所有订阅的节点会合并显示在服务器列表中。你可以在“设置” > “服务器订阅”中管理（编辑或删除）已添加的各个订阅。

### 导入订阅会泄露我的密码或其他信息吗？
一个标准的订阅链接本身只包含服务器配置信息，不会直接泄露你的主账户密码。但是，订阅链接本身是一串密钥，**任何人获得此链接都可以使用其对应的服务**。因此，你必须像保管密码一样保管好你的订阅链接，切勿公开分享或泄露。
