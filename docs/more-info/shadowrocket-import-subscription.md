# 小火箭怎么导入订阅

## 小火箭导入订阅的详细步骤

小火箭（Shadowrocket）是一款在iOS设备上广受欢迎的代理工具，其核心功能之一就是通过导入订阅链接来获取并更新代理服务器节点。这个过程是使用小火箭的第一步，也是关键一步。下面将详细介绍几种主流且安全的导入方法。

### 方法一：通过订阅链接直接导入（最常用）
这是最快捷的方式，适用于您已经拥有一个订阅链接（通常是一串以 `http://` 或 `https://` 开头的URL）。
1.  **复制订阅链接**：从您的服务提供商处获取完整的订阅链接，并确保已将其复制到剪贴板。
2.  **打开小火箭**：在您的iPhone或iPad上启动小火箭应用。
3.  **点击“添加订阅”**：应用首页左上角通常有一个“+”号或直接显示“添加订阅”按钮，点击它。
4.  **粘贴并确认**：在出现的URL输入框中，粘贴您复制的订阅链接。下方的“备注”可以填写一个方便您识别的名称（如“我的服务”）。
5.  **完成导入**：点击右上角的“完成”，小火箭会自动开始从该链接下载节点配置。下载成功后，所有节点会出现在首页的服务器列表中。

### 方法二：通过二维码扫描导入
许多服务商提供二维码形式的订阅，这种方式可以避免手动输入错误，更为方便。
1.  **打开扫描功能**：在小火箭首页，点击右上角的“扫描”图标（一个二维码形状的按钮）。
2.  **授权相机权限**：如果是首次使用，请允许小火箭访问您的相机。
3.  **扫描二维码**：将手机摄像头对准服务商提供的订阅二维码。
4.  **自动处理**：扫描成功后，小火箭会自动识别二维码中的订阅信息，并弹出确认窗口。您只需点击“完成”或“确定”即可导入节点。

### 方法三：通过配置文件导入
有时您可能会获得一个 `.conf` 或 `.yaml` 等格式的配置文件。
1.  **获取配置文件**：将配置文件保存到您的iOS设备“文件”App中，或通过其他应用（如邮件、Safari浏览器）下载。
2.  **用其他应用打开**：在文件所在的应用中，找到该配置文件，点击“分享”或“用其他应用打开”按钮。
3.  **选择小火箭**：在分享菜单的应用列表中，找到并选择“拷贝到‘小火箭’”。
4.  **导入成功**：小火箭会自动打开并导入配置，节点将添加到您的列表中。

**重要提示**：
*   **订阅来源**：请务必从可信赖的服务商处获取订阅，以保障网络安全和连接稳定性。
*   **更新订阅**：服务器节点可能会更新或变动。您可以长按首页的订阅名称，选择“更新”来手动获取最新节点。建议定期更新。
*   **兼容性**：确保您的订阅链接或配置文件与小火箭的协议兼容。

如果您是初次在苹果设备上使用此类工具，对于安装和基础操作还有疑问，可以参考我们的详细指南：[MAC&IOS下载以及使用方式](/network/vpn/how-to-download-and-install-tools/mac-ios-download-usage)，其中涵盖了从获取应用到初步设置的完整流程。

## 常见问题

### 导入订阅后，节点列表是空的怎么办？
这通常有几个原因：
1.  **订阅链接失效**：您复制的链接可能已过期或错误。请返回服务商后台检查并重新复制正确的链接。
2.  **网络问题**：首次导入时，需要设备能正常访问订阅链接所在的服务器。请检查当前网络连接，或尝试切换网络（如从Wi-Fi切换到蜂窝数据）后，长按订阅名称选择“更新”。
3.  **格式不支持**：极少数情况下，订阅内容格式不被小火箭解析。请联系您的服务商确认其提供的订阅是否支持Shadowrocket客户端。

### 如何更新和删除订阅？
*   **更新订阅**：在小火箭首页，找到您已添加的订阅名称，**向左滑动**它，就会出现“更新”选项。点击即可手动更新节点。您也可以点击订阅名称进入详情页，在底部找到“更新”按钮。
*   **删除订阅**：同样，在订阅名称上**向左滑动**，会出现“删除”选项。点击即可移除整个订阅及其下的所有节点。如果您只想删除某个具体节点，可以在节点列表中对单个节点执行左滑删除操作。
