# shadowrocket导入订阅

## Shadowrocket如何导入订阅链接

Shadowrocket是一款在iOS平台上广受欢迎的代理工具，它支持多种代理协议，如SS、SSR、V2Ray、Trojan等。其核心功能之一就是通过“订阅”来便捷地获取和管理代理服务器节点。导入订阅是使用Shadowrocket的第一步，也是关键操作。

## 订阅链接的获取与准备

在开始导入前，您首先需要拥有一个可用的订阅链接。这个链接通常由您所购买的VPN或代理服务提供商提供。它是一串以 `http://` 或 `https://` 开头的URL。请务必从可信赖的服务商处获取，并妥善保管，不要随意泄露。

**重要提示**：订阅链接是您获取节点列表的凭证，它本身不直接提供代理服务，而是包含了服务器地址、端口、密码和加密方式等信息的加密列表。

## 详细导入步骤

请按照以下步骤在Shadowrocket中完成订阅导入：

1.  **打开Shadowrocket**：确保您已在App Store下载并安装了正版Shadowrocket应用。
2.  **进入订阅页面**：打开应用后，点击底部导航栏的 **“设置”** 图标（齿轮状），然后选择 **“服务器订阅”**。
3.  **添加订阅**：点击右上角的 **“+”** 号按钮。
4.  **填写订阅信息**：
    *   **URL**：在此处粘贴您从服务商那里获取的完整订阅链接。
    *   **备注**：您可以为此订阅设置一个易于识别的名称，例如“我的工作节点”或服务商名称。
5.  **保存并更新**：填写完毕后，点击右上角的 **“完成”**。
6.  **手动更新节点**：返回“服务器订阅”页面，您会看到刚刚添加的订阅。对其**向左滑动**，会出现“更新”选项，点击它。或者，直接点击该订阅条目，在详情页点击 **“更新”** 按钮。
7.  **节点生效**：更新成功后，所有节点会自动出现在Shadowrocket首页的服务器列表中。此时，您只需在首页选择一个延迟低、速度快的节点，点击右上角的开关即可连接。

### 高级设置与技巧

*   **自动更新**：在“服务器订阅”页面，开启 **“打开时更新”** 选项。这样每次打开Shadowrocket应用时，它会自动检查并更新订阅，确保您始终获得最新的节点信息。
*   **多个订阅管理**：您可以添加多个不同服务商的订阅链接，方便管理和切换。通过左滑订阅条目可以进行“更新”、“编辑”或“删除”操作。
*   **本地文件导入**：除了订阅链接，Shadowrocket也支持导入包含单个节点信息的配置文件（如`.ss`、`.vmess`链接等）。您可以通过“从URL下载”或“从剪贴板导入”等方式添加。

## 导入失败常见原因与排查

如果导入后无法更新节点，请检查以下几点：
1.  **网络连接**：确保您的设备可以正常访问互联网，尝试切换Wi-Fi或蜂窝数据。
2.  **链接有效性**：确认订阅链接没有过期，并且已从服务商后台正确复制了完整链接。
3.  **应用权限**：首次使用时，请确保允许Shadowrocket使用本地网络和VPN配置。
4.  **系统时间**：检查设备的系统日期和时间是否准确，错误的时间可能导致SSL证书验证失败。

## 常见问题

### 为什么更新订阅后节点列表是空的？
这通常意味着订阅链接本身无效、已过期，或者在更新时网络连接出现了问题。请首先检查您的网络，然后重新向服务商确认订阅链接是否有效，并尝试再次手动更新。如果问题依旧，请联系您的服务商获取技术支持。

### 可以同时使用多个订阅吗？
是的，完全可以。Shadowrocket支持添加和管理多个订阅。您可以为不同用途（如流媒体、工作、游戏）添加不同服务商的订阅。在连接时，您可以从合并后的总节点列表中任意选择其中一个进行连接。

### 导入订阅安全吗？
只要您从信誉良好的正规服务商处获取订阅链接，并通过Shadowrocket官方应用导入，整个过程是安全的。Shadowrocket本身不会记录或上传您的订阅信息。请避免使用来历不明的免费订阅链接，以防隐私泄露或网络安全风险。
