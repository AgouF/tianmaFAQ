# 小火箭怎么导入订阅

## 小火箭导入订阅详细教程

小火箭（Shadowrocket）是一款在iOS设备上广受欢迎的代理工具，其核心功能之一就是通过导入订阅链接来获取并更新代理服务器节点。这个过程是将您从服务提供商处获得的订阅信息，快速、批量地添加到小火箭客户端中的关键步骤。掌握正确的导入方法，能让您的网络连接体验更加顺畅。

### 准备工作
在开始导入之前，请确保您已完成以下准备：
1.  **已安装小火箭**：确保您的iPhone或iPad上已从App Store下载并安装了最新版本的Shadowrocket应用。如果尚未安装，可以参考我们的详细指南：[MAC&IOS下载以及使用方式](/network/vpn/how-to-download-and-install-tools/mac-ios-download-usage)。
2.  **获取有效的订阅链接**：这是最关键的一步。您需要从您所购买的VPN或代理服务商那里获得一个专属的订阅链接（通常是一个以 `http://` 或 `https://` 开头的URL）。请妥善保管此链接，不要随意泄露。
3.  **保持网络畅通**：导入过程需要联网下载节点信息。

### 主要导入方法
小火箭提供了多种便捷的导入方式，您可以根据自己的习惯选择。

#### 方法一：通过“扫码”导入（最推荐）
如果您的服务商提供了订阅链接的二维码，这是最快捷的方式。
1.  打开小火箭应用，点击应用右上角的“加号”（+）按钮。
2.  在弹出的菜单中，选择“扫描二维码”。
3.  将手机摄像头对准服务商提供的订阅二维码。
4.  扫描成功后，应用会自动识别并弹出确认窗口，点击“确定”即可完成导入。所有节点会自动出现在服务器列表中。

#### 方法二：通过“从URL导入”
如果您只有订阅链接的文本形式，可以使用此方法。
1.  在小火箭主界面，同样点击右上角的“加号”（+）。
2.  选择“从URL导入”。
3.  在弹出的输入框中，完整地粘贴您复制的订阅链接。
4.  点击右上角的“完成”，小火箭会自动下载并解析订阅内容，成功后会提示“成功”。

#### 方法三：通过“剪贴板”导入
如果订阅链接已经复制在剪贴板中，小火箭可以智能识别。
1.  确保订阅链接已复制到系统剪贴板（例如，在Safari中长按链接选择“拷贝”）。
2.  直接打开小火箭应用，通常会立刻弹出一个提示框，询问是否从剪贴板添加服务器。
3.  点击“确定”即可快速导入。

#### 方法四：通过Safari浏览器打开
您也可以在任何地方（如邮件、备忘录、Telegram）直接点击订阅链接。
1.  点击订阅链接，系统会默认在Safari浏览器中打开。
2.  页面加载后，通常会提示“在Shadowrocket中打开链接？”，点击“打开”。
3.  应用跳转到小火箭并自动完成导入。

### 导入后的操作与注意事项
- **更新订阅**：服务器节点信息可能会更新（如密码、地址变更）。您可以在小火箭中，向左滑动已添加的订阅配置，点击“更新”来手动获取最新节点。建议定期更新。
- **选择节点**：导入成功后，所有节点会列在“服务器”页面。点击任意一个节点即可尝试连接。通常建议选择延迟（ping值）较低的节点。
- **安全提醒**：请务必从可信的服务商处获取订阅，不要使用来源不明的免费订阅，以防隐私泄露或网络安全风险。
- **兼容性**：确保您的订阅链接格式与小火箭兼容（通常支持SS、SSR、Vmess、Trojan等协议的订阅标准格式）。

## 常见问题

### 导入订阅后，为什么节点列表是空的？
出现这种情况通常有几个原因：
1.  **订阅链接失效**：您的订阅可能已过期，或者链接本身有误。请向您的服务商确认链接的有效性。
2.  **网络问题**：导入时网络连接不稳定，未能成功下载节点信息。请切换至稳定的网络（如Wi-Fi）后，尝试在订阅配置上左滑选择“更新”。
3.  **链接格式不兼容**：极少数情况下，服务商提供的订阅格式可能不被小火箭直接支持。您可以尝试联系服务商，询问是否有针对Shadowrocket的专用订阅链接。

### 如何更新订阅和删除旧的订阅配置？
- **更新单个订阅**：在小火箭的“服务器”页面，找到您已添加的订阅配置（通常显示为订阅名称或URL开头），向左滑动它，就会出现“更新”和“删除”选项。点击“更新”即可获取最新节点。
- **删除订阅**：同样通过左滑操作，点击“删除”即可移除该订阅配置及其下的所有节点。这不会影响您已建立的其他订阅。定期清理无效或不再使用的订阅是个好习惯。
