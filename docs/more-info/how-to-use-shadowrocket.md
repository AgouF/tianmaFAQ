# 小火箭怎么用

## 小火箭（Shadowrocket）简介

小火箭（Shadowrocket）是一款在iOS平台上广受欢迎的代理工具应用程序，主要用于配置和使用Shadowsocks、V2Ray、Trojan等多种代理协议，帮助用户科学上网、保护网络隐私或访问特定网络资源。它以其简洁的界面、强大的功能和稳定的连接而备受用户青睐。需要注意的是，该应用仅在非中国区的App Store提供下载。

## 核心使用步骤

要使用小火箭，你需要完成以下几个核心步骤：

1.  **获取订阅链接或节点信息**：这是使用任何代理服务的前提。你需要从一个可靠的代理服务提供商那里购买服务，并获得相应的订阅链接（URL）或单个服务器的配置信息（包括服务器地址、端口、密码和加密方式等）。
2.  **添加代理配置**：
    *   **通过订阅链接添加（推荐）**：这是最方便的方法。打开小火箭，点击右上角的“+”号，选择“类型”为“Subscribe”。将服务商提供的订阅链接粘贴到“URL”栏中，然后点击右上角的“完成”。小火箭会自动从该链接获取并更新节点列表。
    *   **手动添加节点**：如果没有订阅链接，可以选择“类型”为相应的协议（如Shadowsocks），然后手动填写服务器、端口、密码、加密方式等所有参数。
3.  **连接与断开**：添加配置后，在主界面的“服务器”列表中，你会看到所有可用的节点。点击你想要连接的节点，然后点击界面底部大大的圆形开关按钮，即可开启代理连接。连接成功后，按钮会变为绿色，状态栏会出现小火箭的VPN图标。再次点击开关即可断开连接。
4.  **选择代理模式**：小火箭提供了灵活的代理规则：
    *   **配置**：通常选择“代理”即可，这意味着所有流量都通过你选择的节点。
    *   **代理**：即全局代理模式。
    *   **场景**：可以根据不同的网络环境（如蜂窝数据、Wi-Fi）设置不同的代理模式或节点。
    *   **规则**：这是最常用的模式。在此模式下，只有被规则列表匹配的流量（如访问国外网站）会走代理，访问国内网站则直连，速度更快且节省流量。你可以导入或自定义规则列表。

## 高级功能与使用技巧

*   **测试节点速度**：长按某个节点，在弹出的菜单中选择“测试”，可以ping延迟或进行速度测试，帮助你选择最优节点。
*   **订阅更新**：服务商的节点信息可能会更新。你可以进入“设置”->“服务器订阅”，开启“打开时更新”或手动点击“更新”来获取最新节点。
*   **分流规则（规则配置）**：熟练用户可以在“配置”->“规则”中，导入更强大的规则文件（如来自`github.com`的`DivineEngine`规则），实现更精准和智能的分流。
*   **日志查看**：如果连接出现问题，可以在“日志”选项卡中查看详细的连接信息，有助于排查故障。
*   **分享配置**：你可以将自己的配置（非订阅链接）生成二维码分享给其他小火箭用户。

## 常见问题

### 小火箭在哪里下载？
小火箭（Shadowrocket）目前仅在美国区等非中国大陆的Apple App Store上架。你需要一个相应地区的Apple ID才能搜索、购买和下载。请勿从不明来源的第三方网站下载安装包，以防安全风险。

### 连接上了但无法上网怎么办？
这是一个常见问题，请按顺序排查：
1.  **检查节点状态**：确认你的代理服务套餐是否在有效期内，以及节点是否可用。可以尝试切换其他节点。
2.  **检查代理模式**：如果你使用的是“规则”模式，可能是规则列表不匹配。可以暂时切换到“代理”（全局）模式测试。如果能上网，说明是规则问题。
3.  **检查本地网络**：尝试切换Wi-Fi和蜂窝数据，或重启飞行模式。
4.  **更新订阅**：在“设置”->“服务器订阅”中手动更新订阅，确保节点信息是最新的。

### 小火箭和Clash有什么区别？
两者都是优秀的代理工具，但设计哲学不同。小火箭更偏向于一个“客户端”，配置相对直观，核心是连接和管理节点。Clash（及其衍生客户端如Stash、ClashX）则更侧重于“规则引擎”，拥有极其强大和复杂的规则组、策略组功能，适合对网络分流有极高要求的进阶用户。对于大多数普通用户，小火箭的易用性已经足够。
