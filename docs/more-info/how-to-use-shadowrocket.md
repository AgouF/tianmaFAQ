# 小火箭怎么用

## 小火箭入门指南

小火箭（Shadowrocket）是一款在iOS平台上广受欢迎的代理工具，它支持多种代理协议，可以帮助用户科学上网、访问国际网络和保护隐私。对于初次使用的用户来说，掌握其基本使用方法至关重要。

### 核心功能与配置原理
小火箭的核心功能是通过配置代理服务器，将您的网络流量安全地转发到目标服务器。它支持SS/SSR、Vmess（V2Ray）、Trojan等主流协议。在使用前，您需要先获取可用的服务器订阅链接或手动配置信息，这些通常由您的VPN服务提供商提供。

配置过程主要分为两步：首先是获取并导入服务器配置，其次是开启全局或规则模式进行连接。小火箭的“规则”功能非常智能，可以自动判断哪些网站走代理，哪些直接连接，从而实现高效且省流量的上网体验。

## 详细使用步骤

### 1. 获取与安装
首先，您需要一个非中国区的Apple ID，在App Store中搜索“Shadowrocket”进行购买和下载。安装完成后，打开应用。

### 2. 添加服务器配置
这是最关键的一步。您通常有两种方式添加服务器：
*   **通过订阅链接添加（推荐）**：点击应用首页右上角的“+”号，选择“类型”为“Subscribe”。在“URL”栏中粘贴您的服务商提供的订阅链接，然后点击右上角“完成”。小火箭会自动从该链接获取并更新服务器列表。
*   **手动配置**：同样点击“+”号，根据服务商提供的单节点信息，手动填写服务器地址、端口、密码、加密方式等。这种方式适合只有单个服务器节点的用户。

### 3. 连接与使用
添加服务器后，您会在首页看到服务器列表。点击您想连接的服务器，然后点击顶部大大的开关按钮，即可开启代理连接。连接成功后，开关会变为绿色，状态栏会出现小火箭的VPN图标。

*   **模式选择**：
    *   **配置模式**：默认且最常用的模式。它会根据您设定的规则（如绕过中国大陆地址、广告拦截等）智能分流。
    *   **代理模式**：所有流量都通过代理服务器。
    *   **直连模式**：关闭代理，所有流量直接连接。
    您可以在首页的“全局路由”中进行切换。

### 4. 高级功能与优化
*   **规则设置**：在“配置”标签页中，您可以管理规则列表。点击“从URL下载配置”可以导入更强大的规则集（如ConnersHua的规则），实现更精准的分流和广告屏蔽。
*   **测试与切换**：长按服务器节点，可以选择“测试URL”来检测该节点的延迟和速度，方便您选择最优线路。
*   **日志查看**：如果遇到连接问题，可以在“日志”标签页中查看详细的连接信息，帮助排查故障。

## 常见问题

### 小火箭连接失败怎么办？
首先，请检查您的网络是否正常。其次，确认服务器配置信息（如地址、端口、密码）是否准确无误且未过期。如果使用订阅链接，尝试“更新订阅”。如果问题依旧，可以尝试切换不同的代理协议（如从V2Ray切换到Trojan）或联系您的服务商确认服务器状态。

### 小火箭和Clash有什么区别？我该用哪个？
小火箭和Clash都是优秀的代理工具，但架构不同。小火箭更偏向于一个功能强大、配置直观的客户端，适合大多数用户，尤其是iOS用户。Clash（及其衍生版本如ClashX for Mac）核心在于其强大的规则引擎和“配置”文件，功能更极客化，支持更复杂的代理策略。对于Mac和iOS用户，可以根据自己的需求选择。如果您希望获得更全面的平台使用指导，可以参考这篇关于[Mac & iOS下载以及使用方式](/network/vpn/how-to-download-and-install-tools/mac-ios-download-usage)的文章，其中涵盖了不同工具的选择和基础配置。

### 小火箭会影响手机耗电吗？
开启代理连接会持续占用网络和少量计算资源，因此会比不开启时略微增加耗电，但这在正常可接受范围内。为了节省电量，建议在不需要时及时关闭顶部的代理开关。使用“配置模式”并搭配良好的规则，可以让国内应用直连，也能有效减少不必要的代理流量和电量消耗。
