# 小火箭怎么用

## 小火箭入门指南

小火箭（Shadowrocket）是一款在iOS平台上广受欢迎的代理工具，它功能强大且配置灵活，可以帮助用户科学上网、访问国际网络和保护隐私。对于初次使用的用户来说，掌握其基本使用方法至关重要。

### 下载与安装
首先，你需要拥有一个非中国区的Apple ID（例如美区、港区等），才能在App Store中搜索并购买下载“Shadowrocket”。请注意，这是一款付费应用。下载完成后，你可以在主屏幕上找到它的图标。

### 基础配置与使用
成功安装后，打开小火箭应用。其核心功能是添加并启用代理服务器（节点）。这些节点信息通常由你所订阅的VPN服务提供商提供。

1.  **添加节点**：点击应用右上角的“+”号，你可以通过多种方式添加节点：
    *   **扫描二维码**：这是最方便的方式。如果你的服务商提供了节点二维码，直接点击“扫描二维码”即可自动导入。
    *   **手动输入**：选择“类型”为你的节点协议（如SS、SSR、V2Ray、Trojan等），然后逐一填写服务器、端口、密码、加密方式等信息。
    *   **订阅链接**：许多服务商提供订阅链接。点击“订阅”，然后添加订阅链接URL，小火箭会自动获取并更新节点列表。

2.  **连接与断开**：添加节点后，它会在主界面列表中显示。点击你想要连接的节点，然后点击顶部大大的开关按钮，即可建立连接。连接成功后，开关会变为绿色，状态栏会出现VPN图标。再次点击开关即可断开连接。

3.  **全局路由与代理规则**：这是小火箭的智能之处。在首页底部，你可以设置“配置”下的“代理”规则。
    *   **代理**：所有流量都通过代理服务器。
    *   **配置**（推荐）：这是默认且最常用的模式。它会根据你配置的规则文件（如`default.conf`）来决定哪些网站走代理，哪些直连（比如国内网站）。这能有效节省流量并提升访问国内网站的速度。
    *   **直连**：所有流量都不经过代理。

### 高级功能与优化
*   **规则配置**：点击底部的“配置”标签，你可以导入或编辑规则文件，实现更精细化的分流（例如让Netflix走美国节点，其他流量走日本节点）。
*   **延迟测试**：长按节点列表中的某个节点，可以选择“测试真连接延迟”，这比普通的ICMP Ping更能反映实际代理速度。
*   **HTTPS解密**（谨慎使用）：此功能可用于调试或过滤广告，但需要安装并信任证书，存在一定安全风险，普通用户不建议开启。

### 使用注意事项
*   确保你的节点信息准确有效。
*   在“配置”模式下，初次使用可能需要从网络下载规则文件，请保持网络通畅。
*   注意手机的电量消耗，长期开启代理可能会略微增加耗电。
*   遵守当地法律法规，合理使用工具。

## 常见问题

### 小火箭连接后无法上网怎么办？
首先检查节点信息是否正确，以及节点本身是否可用。其次，检查顶部的“全局路由”设置，如果你设置为“配置”模式，但规则文件有问题或缺失，也可能导致无法上网。可以尝试切换到“代理”模式测试。如果问题依旧，请尝试重启小火箭应用或手机。

### 小火箭和Clash有什么区别？我该用哪个？
两者都是优秀的代理工具。小火箭更偏向于一个功能全面的客户端，配置直观，适合手动管理节点和简单规则。Clash（及其衍生客户端如Stash）核心是基于YAML配置文件的规则引擎，分流规则更强大、灵活，适合喜欢“一次配置，自动分流”的进阶用户，但上手门槛稍高。对于iOS新手用户，从小火箭开始更容易。如果你需要更复杂的多节点负载均衡和精细规则，可以后续探索Clash。
