# 小火箭怎么用

## 小火箭（Shadowrocket）是什么？

小火箭（Shadowrocket）是一款在iOS平台上广受欢迎的代理工具客户端。它支持多种代理协议，如 Shadowsocks、V2Ray、Trojan、HTTP/HTTPS/SOCKS5 等，允许用户通过配置服务器信息，安全、便捷地访问国际网络或保护本地网络隐私。其界面简洁，功能强大，是许多iPhone和iPad用户管理网络连接的首选工具。

## 如何获取与安装小火箭

由于小火箭在中国大陆的App Store中无法直接下载，用户通常需要通过其他地区的Apple ID账号（如美区、港区）登录App Store进行购买和下载。获取账号后，在App Store中搜索“Shadowrocket”即可找到并安装。

**请注意**：请务必通过官方正规渠道获取应用，以确保账号安全和软件稳定性。关于更详细的下载和安装步骤，特别是对于Mac和iOS新用户，您可以参考我们的专门指南：[MAC&IOS下载以及使用方式](/network/vpn/how-to-download-and-install-tools/mac-ios-download-usage)。

## 基础使用教程：配置你的第一个代理

安装完成后，按照以下步骤进行基本配置：

1.  **获取服务器信息**：首先，你需要拥有一个可用的代理服务器。这通常由服务提供商（VPN/代理服务商）提供，信息包括服务器地址、端口、密码、加密方式以及协议类型（如SS、V2Ray等）。
2.  **添加服务器配置**：
    *   打开小火箭应用，点击右上角的“+”加号按钮。
    *   根据你获得的服务器信息，在“类型”中选择对应的协议（例如 Shadowsocks）。
    *   依次填写“服务器”（IP地址或域名）、“端口”、“密码”和“加密方法”。
    *   可以为此配置命名（如“我的服务器”），然后点击右上角的“完成”保存。
3.  **连接与断开**：在服务器列表页面，找到你刚添加的配置，点击右侧的开关按钮，即可连接。连接成功后，屏幕顶部会出现一个VPN小图标，表示流量正在通过代理服务器。再次点击开关即可断开连接。

## 进阶功能与使用技巧

小火箭提供了许多高级功能以优化使用体验：

*   **全局路由**：在“配置”页面，可以设置“代理”模式（所有流量走代理）、“配置”模式（根据规则分流）或“直连”模式。推荐使用“配置”模式配合规则列表，实现国内外网站智能分流（国内直连，国外代理），节省流量并提升访问速度。
*   **规则设置与订阅**：这是小火箭的核心功能之一。你可以手动添加规则（如`DOMAIN-SUFFIX,google.com,Proxy`），但更高效的方式是使用“规则订阅”URL。许多服务商提供规则订阅链接，添加后可以自动更新庞大的分流规则库。
*   **HTTPS解密**：此功能可用于调试网络请求，但普通用户不建议开启，因为它需要安装并信任证书，存在一定安全风险。
*   **测速与延迟**：长按服务器列表中的某个配置，可以选择“测试真连接延迟”，帮助你选择当前网络环境下速度最快的服务器。

## 常见问题

### 小火箭连接后无法上网怎么办？
请按顺序排查：
1.  **检查服务器信息**：确认填写的地址、端口、密码和加密方式完全正确。
2.  **检查网络**：确保你的设备本身可以正常访问互联网（如蜂窝数据或Wi-Fi已开启）。
3.  **切换连接模式**：尝试在“配置”中，将“全局路由”临时切换到“代理”模式，测试是否能上网。如果可以，则可能是规则配置问题。
4.  **联系服务商**：服务器可能暂时故障或已过期，请联系你的代理服务提供商确认服务器状态。

### 如何实现国内外网站智能分流？
最便捷的方法是使用规则订阅功能。
1.  在小火箭主界面，点击底部的“配置”标签。
2.  点击“规则”部分下的“编辑配置”。
3.  找到“规则”一栏，点击进入后，点击右上角的“+”号，选择“添加规则集”。
4.  在“URL”处粘贴服务商提供的规则订阅链接（或网络上可靠的GFWList分流规则链接），然后点击“下载”。
5.  下载完成后，确保在“全局路由”中选择了“配置”模式。这样，访问国内网站时会直连，访问被屏蔽的国外网站时会自动走代理。
