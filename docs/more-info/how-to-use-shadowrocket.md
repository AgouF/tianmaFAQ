# 小火箭怎么用

## 小火箭（Shadowrocket）简介

小火箭（Shadowrocket）是一款在iOS平台上广受欢迎的代理工具应用程序，主要用于配置和使用Shadowsocks、V2Ray、Trojan等多种代理协议，帮助用户科学上网，访问国际网络并保护网络隐私。它以其简洁的界面、强大的功能和稳定的连接而著称。需要注意的是，小火箭仅在非中国区的App Store提供下载，用户需要使用相应的Apple ID才能获取。

## 如何配置小火箭

成功安装小火箭后，配置是使用的关键。通常，你需要从可靠的代理服务提供商那里获取服务器配置信息。配置方式主要分为以下两种：

1.  **手动配置**：
    *   打开小火箭应用，点击右上角的“+”号。
    *   根据你订阅的服务类型，选择相应的类型，如“Shadowsocks”、“V2Ray”或“Trojan”。
    *   手动逐项填写服务器地址（Host）、端口（Port）、密码（Password）、加密方式（Encryption）等信息。这种方法适合拥有独立服务器信息的用户。

2.  **订阅链接/扫码配置（推荐）**：
    *   这是最常用和便捷的方式。你的服务商通常会提供一个订阅链接（URL）或一个配置二维码。
    *   **订阅链接**：在小火箭首页，点击右上角的“+”号，选择“Subscribe”（订阅），然后将链接粘贴到“URL”栏中，点击“完成”即可自动获取并导入所有服务器节点。
    *   **扫码配置**：在小火箭首页，点击右上角的扫描图标，直接扫描服务商提供的二维码，配置会自动完成。

配置完成后，服务器节点会出现在首页列表中。

## 基本使用与高级功能

配置好节点后，使用就非常简单了：
*   在首页的服务器列表中，点击你想要连接的节点。
*   然后点击顶部大大的开关按钮，即可开启代理连接。当按钮变为绿色且显示“已连接”时，表示连接成功。
*   你可以通过iPhone/iPad的通知中心Widget或桌面小组件快速开关代理，非常方便。

小火箭还提供了一些实用高级功能：
*   **代理规则**：在“配置”页面，你可以设置代理规则。默认的“配置”模式（即规则模式）会根据你设定的规则（如绕过中国大陆IP、直连局域网）智能分流，国内网站直连，国外网站走代理，节省流量且速度快。
*   **全局模式**：所有网络流量都通过代理服务器转发。
*   **场景配置**：可以为不同的网络环境（如蜂窝数据、特定Wi-Fi）设置不同的代理模式。
*   **HTTPS解密**：用于调试和分析网络流量（需安装并信任证书）。

## 常见问题

### 小火箭在哪里下载？
小火箭（Shadowrocket）在中国大陆的App Store无法下载。你需要注册一个非中国区的Apple ID（如美国、香港等），登录App Store后即可搜索并下载。请注意账号安全，从官方渠道获取应用。关于更详细的下载和安装步骤，你可以参考这篇指南：[MAC&IOS下载以及使用方式](/network/vpn/how-to-download-and-install-tools/mac-ios-download-usage)。

### 连接上了但无法上网怎么办？
首先，请检查以下几点：
1.  **节点状态**：确认你的代理服务订阅是否有效，服务器是否正常运行。
2.  **本地网络**：确保你的设备已成功连接Wi-Fi或蜂窝数据。
3.  **配置信息**：核对服务器地址、端口、密码等是否填写正确，特别是使用手动配置时。
4.  **规则设置**：尝试在“配置”页面临时切换到“全局代理”模式，如果全局模式下可以上网，则可能是规则列表（如绕过大陆IP的规则）需要更新或存在问题。
5.  **重启应用**：尝试完全关闭小火箭后重新打开。

### 小火箭和Clash有什么区别？
两者都是优秀的代理工具，但设计理念不同。小火箭更偏向于一个简洁、高效的客户端，核心功能是连接和管理代理节点，上手快速。而Clash（及其衍生客户端如Stash、Clash for Windows）则更侧重于复杂的规则配置和策略组，支持规则脚本、自动测速、负载均衡等更高级的网络策略，适合喜欢深度定制和复杂分流规则的用户。对于大多数普通用户来说，小火箭的易用性已经足够。
