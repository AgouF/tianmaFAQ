# 小火箭怎么用

## 小火箭是什么？

小火箭（Shadowrocket）是一款在 iOS 和 macOS 平台上广受欢迎的代理工具客户端。它本身并非一个 VPN 服务，而是一个功能强大的网络调试工具和代理客户端，支持多种代理协议。用户可以通过它来配置和管理自己的代理服务器，从而实现科学上网、访问国际网络、保护隐私或进行网络调试等目的。由于其配置灵活、界面直观，它成为了许多苹果设备用户的首选工具。

## 如何获取与安装小火箭

由于小火箭的功能特性，它在中国大陆的 App Store 中无法直接下载。通常，用户需要一个非中国大陆地区的 Apple ID 才能访问 App Store 并下载此应用。

1.  **获取非大陆区 Apple ID**：你可以自行注册一个（例如美区、港区账号），或者从可信的渠道获取。注册时需要准备相应的支付方式和地址信息。
2.  **切换 App Store 账户**：在 iOS 设备的 App Store 或 macOS 的 App Store 中，退出当前的中国区账户，登录你准备好的非大陆区账户。
3.  **搜索与下载**：在 App Store 中搜索 “Shadowrocket”，找到后点击购买/下载。这是一款付费应用，通常需要支付少量费用。
4.  **安装与信任**：下载完成后，首次打开时，如果出现“未受信任的企业级开发者”提示，你需要前往设备的“设置” > “通用” > “VPN与设备管理”（或“描述文件与设备管理”）中，找到对应的开发者证书并选择“信任”。

关于更详细的下载、安装以及初始设置步骤，你可以参考我们的专门指南：[MAC&IOS下载以及使用方式](/network/vpn/how-to-download-and-install-tools/mac-ios-download-usage)。

## 基础配置与使用方法

安装成功后，你需要获取可用的代理服务器信息（如节点订阅链接或手动配置参数）才能开始使用。

**核心使用流程如下：**

1.  **添加代理服务器**：
    *   **订阅链接（推荐）**：这是最常用的方式。在小火箭主界面，点击右上角的“+”号，选择“订阅”。将你的服务提供商提供的订阅链接粘贴到“URL”栏中，然后点击右上角的“完成”。小火箭会自动从该链接获取服务器列表。
    *   **手动配置**：如果你有单个服务器的详细参数（如地址、端口、密码、加密方式等），可以选择“手动添加”来逐一填写。

2.  **连接与断开**：
    *   添加服务器后，在主界面的服务器列表中，你会看到可用的节点。点击你想连接的节点，右侧会出现一个开关，点击打开即可连接。
    *   连接成功后，屏幕顶部会出现一个“VPN”小图标，表示代理正在运行。再次点击开关即可断开连接。

3.  **全局路由与代理规则**：
    *   这是小火箭的核心功能之一。在底部菜单选择“配置”标签页，你可以设置流量的分流规则。
    *   **代理**：所有流量都通过代理服务器。
    *   **配置**（规则模式）：这是最智能的模式。小火箭会根据内置或你自定义的规则列表，决定哪些流量走代理（如国外网站），哪些流量直连（如国内网站）。初次使用建议选择此模式。
    *   **直连**：所有流量都不经过代理。
    *   **场景**：可以根据不同的网络环境（如不同Wi-Fi）自动切换配置模式。

## 高级功能与设置建议

*   **延迟测试**：长按服务器列表中的某个节点，可以对其进行“连通性测试”，查看当前延迟，帮助你选择最快的节点。
*   **日志查看**：在“日志”标签页，可以查看详细的连接和请求日志，对于排查网络问题非常有帮助。
*   **HTTPS 解密**：高级用户可用于调试 HTTPS 流量，普通用户无需开启。
*   **省电与后台刷新**：为确保应用在后台持续运行，建议在 iOS 的“设置” > “Shadowrocket”中，开启“无线数据”和“后台App刷新”。

**使用建议**：对于大多数用户，保持“规则”模式，并定期通过订阅链接更新服务器列表，就能获得稳定、智能的上网体验。请务必从可信赖的服务商处获取订阅链接，以保障网络安全和隐私。

## 常见问题

### 小火箭连接后无法上网怎么办？
首先检查你的代理服务器订阅是否有效，节点信息是否正确。可以尝试切换到“直连”模式，如果能上网，则说明是代理服务器的问题。如果直连也无法上网，请检查你的本地网络连接。此外，可以查看小火箭的“日志”页面，通常会有具体的错误信息提示。

### 小火箭和 VPN 有什么区别？
小火箭是一个**代理客户端工具**，它需要你自行配置可用的代理服务器（这些服务器可能由第三方VPN服务商或机场提供）。而通常所说的 VPN（如一些知名品牌的VPN应用）是一个**集成了客户端和自家服务器网络的服务**，你只需登录账户即可使用。小火箭提供了更灵活的协议支持和分流能力。

### 为什么小火箭在后台经常断开？
这通常是 iOS 系统为了节省电量而强制关闭后台应用所致。请确保在 iOS 系统设置中，为小火箭开启了“后台App刷新”权限。同时，在小火箭的设置里，可以尝试开启“兼容性模式”或“始终连接”等选项来增强稳定性。保持应用在前台运行是最稳定的方式。
