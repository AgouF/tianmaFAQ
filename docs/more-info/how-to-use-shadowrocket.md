# 小火箭怎么用

## 小火箭（Shadowrocket）简介

小火箭（Shadowrocket）是一款在iOS平台上广受欢迎的代理工具客户端，它支持多种代理协议，如SS/SSR、V2Ray、Trojan等，可以帮助用户科学上网、访问被限制的网站或保护网络隐私。其界面简洁，功能强大，但需要用户自行配置节点信息才能使用。对于初次接触的用户来说，了解其基本使用方法至关重要。

## 如何获取与安装小火箭

由于小火箭在中国大陆的App Store中无法直接下载，用户通常需要通过非大陆区的Apple ID（如美区、港区账号）登录App Store进行购买和下载。这是一次性付费应用。下载完成后，即可在iPhone或iPad的主屏幕上看到其图标。

关于更详细的获取途径、账号注册及安装步骤，您可以参考我们的专门指南：[MAC&IOS下载以及使用方式](/network/vpn/how-to-download-and-install-tools/mac-ios-download-usage)，其中包含了针对苹果设备的通用下载和初步设置方法。

## 基础配置与使用步骤

安装成功后，使用小火箭的核心是为其添加可用的代理服务器（节点）信息。

1.  **获取节点订阅链接或配置信息**：您需要从可靠的代理服务提供商处获取订阅链接（Subscription URL）或单个服务器的具体配置参数（如地址、端口、密码、加密方式等）。

2.  **添加节点配置**：
    *   **通过订阅链接添加（推荐）**：这是最方便的方法。打开小火箭，点击底部导航栏的“设置”（齿轮图标），选择“服务器订阅”。点击右上角的“+”号，将服务商提供的订阅链接粘贴到“URL”栏中，然后点击右上角“完成”。返回主界面，点击底部的“连接”页签，再点击右上角的“更新”按钮，即可获取并列出所有节点。
    *   **手动添加单个节点**：在主界面点击右上角的“+”号，根据服务商提供的协议（如SS、V2Ray），手动填写服务器、端口、密码、加密方式等所有详细信息，然后保存。

3.  **连接与断开**：在“连接”页面，从节点列表中选择一个你想使用的节点。然后点击顶部大大的圆形开关按钮，小火箭会尝试连接。连接成功后，开关会变为绿色，顶部状态栏会出现VPN图标。此时，你的设备网络流量即通过该代理服务器传输。要断开连接，再次点击开关按钮即可。

4.  **模式选择**：小火箭提供了灵活的流量分流规则。
    *   **配置模式**：默认模式，根据用户设定的规则（如绕过中国大陆IP）决定哪些流量走代理。
    *   **代理模式**：所有流量都通过代理。
    *   **直连模式**：所有流量都不通过代理。
    您可以在连接页面的节点列表上方进行切换，通常使用“配置模式”即可满足大多数需求。

## 使用技巧与注意事项

*   **规则配置**：在“配置”页面，可以管理分流规则。默认的规则组（如“GFW List”）通常够用，它能自动判断并代理访问被墙的网站，而国内网站则直连，速度更快。
*   **测试连接**：连接后，可以打开浏览器访问 `google.com` 或 `youtube.com` 来测试是否成功。
*   **后台刷新**：为了确保订阅节点能自动更新，建议在iOS系统的“设置” > “通用” > “后台App刷新”中，为小火箭开启此功能。
*   **安全性**：请务必从可信的服务商处获取节点，避免隐私泄露。定期更新订阅以获取新鲜可用的节点。
*   **耗电与网络**：长期开启代理可能会略微增加耗电量。如果遇到某些App无法联网，可以尝试切换到“直连模式”排查是否为代理规则问题。

## 常见问题

### 小火箭连接成功但无法上网怎么办？
首先，请确认您设备本身的蜂窝数据或Wi-Fi网络是通畅的。其次，问题可能出在节点上：尝试切换到节点列表中的其他节点进行连接。最后，检查分流规则，可暂时切换到“代理模式”测试，如果此时能上网，说明是规则配置问题，可能需要更新规则或检查规则列表是否准确。

### 如何更新节点订阅？
进入小火箭的“设置” > “服务器订阅”，找到您已添加的订阅条目，向左滑动，会出现“更新”选项，点击即可手动更新。您也可以开启“自动更新”功能，让小火箭定期自动获取最新节点。

### 小火箭在iPhone和iPad上使用有区别吗？
基本操作和功能完全相同。得益于iOS和iPadOS系统的统一性，一次购买即可在您同一Apple ID下的所有iOS/iPadOS设备上使用。配置信息可以通过iCloud同步，也可以在每台设备上单独添加订阅。
