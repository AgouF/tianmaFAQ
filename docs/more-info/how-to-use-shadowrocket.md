# 小火箭怎么用

## 小火箭（Shadowrocket）简介与核心功能

小火箭（Shadowrocket）是一款在iOS平台上广受欢迎的代理工具应用，主要用于配置和使用VPN、代理服务器等网络连接，以实现科学上网、访问国际网络或增强网络隐私。它支持多种代理协议，如SS/SSR、V2Ray、Trojan、HTTP/Socks5等，因其配置灵活、界面直观而备受用户青睐。要使用它，你首先需要在非中国区的App Store（如美区、港区账号）购买并下载此应用。

## 如何配置和使用小火箭

成功安装应用后，使用小火箭主要分为以下几个步骤：

### 1. 获取服务器订阅链接或手动配置
这是最关键的一步。你需要从可靠的VPN服务提供商那里获取服务器的配置信息。通常有两种方式：
*   **订阅链接（推荐）**：大多数服务商提供订阅URL。你只需复制这个链接。
*   **手动配置**：如果服务商提供了单独的服务器地址、端口、密码和加密方式等信息，你可以手动输入。

### 2. 在App内添加配置
*   打开小火箭应用，点击右上角的“+”号。
*   **使用订阅链接**：选择“类型”为 `Subscribe`，然后将复制的订阅URL粘贴到“URL”栏中，点击右上角“完成”保存。
*   **手动配置**：根据服务器类型（如Shadowsocks），选择对应类型，然后逐一填写服务器、端口、密码、加密方法等信息。

### 3. 连接与断开
添加配置后，你会在主服务器列表看到新增的节点。点击你想要连接的节点，应用会尝试建立连接。连接成功后，屏幕顶部会出现一个VPN图标（小火箭形状），此时你的设备网络流量将通过该代理服务器转发。点击主界面的大圆开关即可断开连接。

### 4. 高级设置与模式选择
小火箭提供了强大的规则配置功能，你可以根据需求设置代理模式：
*   **代理模式**：所有流量都走代理。
*   **配置模式（推荐）**：最常用的模式，只有被规则列表匹配的网站或IP才走代理，其他流量直连。这可以节省国际流量并加速国内访问。
*   **直连模式**：所有流量都不走代理。
你可以在“配置”标签页下的“代理”中编辑规则，或使用服务商提供的规则集链接。

## 使用小火箭的注意事项与技巧

1.  **确保订阅信息有效**：订阅链接或服务器信息可能会过期或失效，如果无法连接，首先检查服务器状态。
2.  **理解代理规则**：合理配置规则是流畅体验的关键。对于新手，可以先使用“代理模式”测试连通性，然后切换到“配置模式”并导入成熟的规则集。
3.  **关注系统通知**：在iOS的“设置” > “Shadowrocket”中，确保开启了“通知”和“VPN”配置权限，否则可能无法正常连接。
4.  **节点选择**：通常选择物理距离近、负载低的节点，可以获得更低的延迟和更快的速度。
5.  **与Clash等工具对比**：小火箭更偏向于简洁直接的代理客户端，而Clash等工具支持更复杂的规则和负载均衡。你可以根据需求选择，相关Mac和iOS平台工具的下载与使用方式，可以参考这篇指南：[MAC&IOS下载以及使用方式](/network/vpn/how-to-download-and-install-tools/mac-ios-download-usage)。

## 常见问题

### 小火箭在哪里下载？
小火箭（Shadowrocket）未在中国大陆的App Store上架。你需要使用一个非中国区的Apple ID（例如美国、香港、日本等地区账号）登录App Store，然后搜索“Shadowrocket”进行购买和下载。请注意，这是一款付费应用。

### 添加订阅后为什么没有服务器节点？
首先，请检查订阅链接是否复制完整且有效。然后，可以尝试在小火箭的“设置” -> “服务器订阅”中，开启“打开时更新”选项，并手动下拉主服务器列表进行刷新。如果仍不显示，可能是订阅链接已失效或格式不被支持，请联系你的服务提供商。

### 连接成功但无法上网怎么办？
请按顺序排查：
1.  检查本地网络是否正常（如Wi-Fi或蜂窝数据）。
2.  尝试切换不同的代理服务器节点。
3.  检查小火箭的“配置” -> “代理”中的规则设置，尝试临时切换到“代理模式”测试。如果代理模式可以上网，说明是规则问题，需要调整或更新规则集。
4.  重启小火箭应用或重启手机。
