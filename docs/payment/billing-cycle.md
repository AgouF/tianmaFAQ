# 账单周期说明

了解我们的计费方式和账单周期。

## 计费方式

| 方案 | 周期 | 说明 |
|------|------|------|
| 月付 | 每月 | 按月扣费，随时可取消 |
| 年付 | 每年 | 一次性支付，享受 **8 折优惠** |
| 按量计费 | 实时 | 根据实际使用量计费 |

## 扣费时间

- **月付**：每月订阅日自动扣费
- **年付**：每年订阅日自动扣费
- **首次订阅**：立即扣费

::: tip 省钱技巧
选择年付方案可节省 **20%** 的费用，推荐长期使用的用户选择。
:::

## 查看账单

1. 进入 **「设置」** → **「账单」**
2. 查看历史账单记录
3. 点击任意账单可下载 PDF 发票

## 取消订阅

取消订阅后，服务将在当前计费周期结束时停止。已付费用不予退还。

::: warning 注意
取消订阅前请确保已备份所有重要数据。
:::
